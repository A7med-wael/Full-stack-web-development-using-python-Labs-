a. Use <title> tag

The <title> tag defines the title of the page shown in the browser tab and search engine results.
It is important for SEO because search engines use it to understand the content of your page, and a good title improves click-through rates and ranking.

b. Use meta charset tag (Why?)

The <meta charset="UTF-8"> tag sets the character encoding for the page.
UTF-8 supports Arabic, English, and many other characters.
Without it, some characters may display incorrectly.

c. Use meta viewport tag (Why?)

The <meta name="viewport"> tag makes the page responsive.
It ensures the content scales properly on different screen sizes, especially on mobile devices.

d. Use meta description tag

The <meta name="description"> tag provides a brief summary of the page content.
It appears in search engine results under the page title.
It helps users understand what the page is about and improves SEO by making search listings more informative and attractive.

e. Set a favicon for your page

A favicon is the small icon that appears in the browser tab beside the page title.
It improves branding and gives your website a professional appearance.
